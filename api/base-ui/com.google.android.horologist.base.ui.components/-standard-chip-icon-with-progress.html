A default icon implementation to be used with a StandardChip that accepts an icon as slot. This implementation displays an icon with a circular progress indicator around it. The progress indicator express the proportion of completion of an ongoing task.

Parameters

progress

The progress of this progress indicator where 0.0 represents no progress and 1.0 represents completion. Values outside of this range are coerced into the range 0..1.

largeIcon

True if it should display the icon with in a large size.

The color of the progress indicator that is around the icon.

The color of the background for the progress indicator.
