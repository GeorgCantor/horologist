A screen to prompt users to sign in.

Through the message the sign-in benefits are explained to the user.

The viewModel will take care of streamlining the process when the user is already signed in. onAlreadySignedIn should be used to navigate away from this screen in that scenario.

The content should provide sign-in alternatives.
